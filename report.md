# SDR-FM-Demodulation Report

## Objective
To learn SDR fundamentals by processing IQ recordings using Python.

## Concepts Used
- IQ sampling and complex baseband
- FFT spectrum analysis
- FM demodulation using phase difference method
- Low-pass filtering
- WAV audio export

## Results
- FFT spectrum plotted (spectrum.png)
- Audio output generated (outputs/output.wav)

## Next Improvements
- Use RTL-SDR to capture real FM broadcast
- Demodulate real music
- Decode RDS (station name/text)
